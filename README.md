# 🧧 老頑童家族 賓果大樂透

過年闔家歡樂的賓果遊戲！老少咸宜，打開瀏覽器就能玩。

## 🎮 遊戲模式

### 🏠 本地模式（單機共享）
- 適合：全家圍著同一個螢幕
- 特點：不需要網路，完全離線可用
- 玩法：輸入所有玩家名字，切換查看各自的賓果牌

### 📡 線上模式（多裝置同步）
- 適合：每個人用自己的手機/平板
- 特點：即時同步開獎進度，自動標記
- 玩法：主持人建立房間，其他人輸入房間碼加入
- **需要：設定 Firebase**（免費，詳見 [FIREBASE_SETUP.md](FIREBASE_SETUP.md)）

## 🎮 遊戲玩法

### 本地模式
1. **輸入玩家名字** — 用頓號分隔，例如：`爸爸、媽媽、阿嬤、小明`
2. **開獎** — 每輪隨機抽出一個過年圖案
3. **自動標記** — 開出的圖案會自動標記在所有玩家的牌上
4. **切換查看** — 用上方的 Tab 切換查看每位玩家的牌面
5. **連線賓果** — 橫、直、斜任一條線連滿 5 格就賓果！🎆

### 線上模式
1. **主持人建立房間** — 輸入名字，點「建立房間」，取得房間碼
2. **其他人加入** — 選「加入房間」，輸入房間碼和自己的名字
3. **主持人開獎** — 只有主持人可以開獎
4. **自動同步** — 所有人的手機自動標記，自動判定連線
5. **賓果慶祝** — 先連線的人看到煙火動畫！🎆

## ✨ 特色

- 🎴 每位玩家的牌面排列都不同（根據名字生成）
- 🏮 滿滿過年年味的視覺設計
- 📱 手機、平板、電腦都能玩
- 🔌 本地模式完全離線可用
- 📡 線上模式支援多裝置即時同步
- 🧧 35 種過年主題圖案
- 🎆 賓果時有煙火慶祝動畫
- 🆓 完全免費（Firebase 免費額度夠用）

## 🚀 快速開始

### 方案一：本地模式（最簡單）

1. 打開遊戲：**https://jameslu8.github.io/cny-bingo/**
2. 選擇「🏠 本地模式」
3. 輸入所有玩家名字（用頓號分隔）
4. 開始遊戲！

**適合：** 全家圍著電視/平板一起玩

### 方案二：線上模式（多裝置）

**第一步：設定 Firebase**（只需設定一次）
1. 參考 [FIREBASE_SETUP.md](FIREBASE_SETUP.md) 完成設定（約 5 分鐘）
2. 更新 `index.html` 中的 Firebase 配置
3. 提交並推送到 GitHub

**第二步：開始遊戲**
1. 主持人打開遊戲，選擇「📡 線上模式」
2. 建立房間，取得房間碼（例如：`ABC123`）
3. 其他人打開遊戲，選「加入房間」，輸入房間碼
4. 主持人點「開獎」，所有人自動同步！

**適合：** 每個人用自己的手機玩

### QR Code 分享

打開 `qrcode.html` 可以產生遊戲連結的 QR Code，方便過年時讓大家掃碼加入。

## 🖥️ 建議使用方式

### 本地模式
- **電視/投影模式**：一台電腦接電視，全家一起看大螢幕
- **平板模式**：一台平板，大家輪流查看各自的牌

### 線上模式
- **手機模式**：每個人用自己的手機，即時同步開獎
- **混合模式**：主持人用電視開獎，其他人用手機看自己的牌
- **遠距模式**：不在同一個地方也能一起玩（視訊 + 線上房間）

## 🛠️ 部署到 GitHub Pages

### 基本部署（僅本地模式）

1. Fork 或 clone 這個 repo
2. 到 **Settings → Pages → Source** 選擇 `main` branch
3. 等待幾分鐘，即可透過 `https://你的帳號.github.io/cny-bingo/` 訪問
4. 開始玩本地模式！

### 啟用線上模式（可選）

5. 參考 [FIREBASE_SETUP.md](FIREBASE_SETUP.md) 設定 Firebase
6. 更新 `index.html` 中的 Firebase 配置
7. 提交並推送變更
8. 等待 GitHub Pages 重新部署
9. 現在可以使用線上模式了！

**提示：** 不設定 Firebase 也能玩，只是沒有多裝置同步功能
