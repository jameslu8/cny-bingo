# 🧧 新春賓果大樂透

過年闔家歡樂的賓果遊戲！老少咸宜，打開瀏覽器就能玩。

## 🎮 遊戲玩法

1. **輸入玩家名字** — 用頓號分隔，例如：`爸爸、媽媽、阿嬤、小明`
2. **開獎** — 每輪隨機抽出一個過年圖案（紅包、燈籠、水餃、舞獅...共 35 種）
3. **自動標記** — 開出的圖案會自動標記在所有玩家的牌上
4. **連線賓果** — 橫、直、斜任一條線連滿 5 格就賓果！🎆
5. **切換查看** — 用上方的 Tab 切換查看每位玩家的牌面

## ✨ 特色

- 🎴 每位玩家的牌面排列都不同（根據名字生成）
- 🏮 滿滿過年年味的視覺設計
- 📱 手機、平板、電腦都能玩
- 🔌 完全離線可用，不需要網路
- 🧧 35 種過年主題圖案
- 🎆 賓果時有煙火慶祝動畫

## 🚀 開始玩

### 線上遊玩

直接打開：**https://你的帳號.github.io/cny-bingo/**

### QR Code 分享

打開 `qrcode.html` 可以產生遊戲連結的 QR Code，方便過年時讓大家掃碼加入。

### 本地遊玩

直接用瀏覽器打開 `index.html` 即可。

## 🖥️ 建議使用方式

- **電視/投影模式**：一台電腦接電視，全家一起看大螢幕
- **手機模式**：分享連結或掃 QR Code，每人用自己的手機開啟
- **平板模式**：一台平板大家輪流查看

## 🛠️ 部署到 GitHub Pages

1. Fork 或 clone 這個 repo
2. 到 **Settings → Pages → Source** 選擇 `main` branch
3. 等待幾分鐘，即可透過 `https://你的帳號.github.io/cny-bingo/` 訪問
4. 記得把 `qrcode.html` 裡的連結改成你自己的 GitHub Pages 網址
